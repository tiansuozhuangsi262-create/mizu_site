# サイト仕様書

## 1. サイト概要

このサイトは、水転写技術を提供する企業「Mizu Site」のウェブサイトです。
製品・サービス、施工事例、お問い合わせフォームなどを通じて、顧客への情報提供とビジネス機会の創出を目的としています。

### ターゲットユーザー
* 水転写技術に興味のある企業
* 製品の加飾を検討している製造業者
* 技術相談を希望する個人・法人

## 2. 主な機能

### 2.1. トップページ (`index.html`)
* サイト全体の概要と主要コンテンツへの誘導
* 企業理念、強み、最新情報などを表示

### 2.2. サービスページ (`service.html`)
* 提供する水転写サービスの詳細説明
* 各サービスの特徴、適用事例などを紹介

### 2.3. 施工事例ページ (`works.html`)
* 過去の水転写施工事例の紹介
* 写真や説明を通じて実績をアピール

### 2.4. お問い合わせページ (`contact.html`)
* 製品やサービスに関するお問い合わせフォーム
* Netlify Forms を利用したデータ収集

### 2.5. その他の機能
* モバイルフレンドリーなレスポンシブデザイン
* ハンバーガーメニューによるナビゲーション（JavaScriptで制御）

## 3. 技術スタック

*   **HTML**: ウェブページの構造定義
*   **CSS**: スタイリング (Tailwind CSS を主に利用)
    *   **カラーパレット**: 青系を基調としたブランドカラー
    *   **フォント**: Google Fonts などを使用
*   **JavaScript**: 動的なコンテンツとフォーム処理
    *   ナビゲーションメニューの開閉
    *   お問い合わせフォームのAJAX送信処理
    *   ページトップへのスムーズスクロール

## 4. デプロイ方法

*   **プラットフォーム**: Netlify
*   **ソースコード管理**: Git (GitHub, GitLab などに連携)
*   **デプロイプロセス**: Gitリポジトリへのプッシュをトリガーとした自動デプロイ
*   **カスタムドメイン**: `ddp-hydro.com` を使用

## 5. フォーム機能

*   **プラットフォーム**: Netlify Forms
*   **フォーム名**: `contact`
*   **スパム対策**: `netlify-honeypot` フィールドを導入
*   **ファイルアップロード**: `enctype="multipart/form-data"` を使用
*   **メール通知**: Netlifyの通知機能を利用。項目名は英語で送信されるが、Netlifyダッシュボードでメールテンプレートをカスタマイズすることで日本語表示が可能。
    *   **注意**: メール本文のレイアウトを詳細にカスタマイズする場合は、Netlify FunctionsまたはZapierなどの外部サービスとの連携が必要。

*   **Google Search Console**: 登録済み (`google-site-verification` TXTレコードによるドメイン所有権確認: `google-site-verification=DAuudNcslqNA0lsBZY2HDSimgwjyV_t2zAK7etwS6Fg`)
*   **サイトマップ**: 今後、XMLサイトマップの生成と登録を検討。
*   **メタタグ**: 各ページの `<head>` セクションに適切な `description` および `keywords` メタタグを設定済み。
*   **セマンティックHTML**: HTML5のセマンティックタグ (`<header>`, `<nav>`, `<main>`, `<footer>` など) を使用して、コンテンツ構造を明確にしています。
*   **レスポンシブデザイン**: モバイルフレンドリーなデザインを採用しており、様々なデバイスで閲覧可能です。

## 7. 画像管理

*   **ディレクトリ構造**: `image/` ディレクトリ内にロゴやその他の主要画像を配置
*   **画像形式**: 主に `png`, `jpg` を使用
*   **最適化**: 画像はウェブ向けに最適化されており、読み込み速度に配慮されています。

## 8. メンテナンスと更新

*   **コンテンツ更新**: HTMLファイルを直接編集して行います。
*   **機能追加・修正**: 必要に応じてHTML, CSS, JavaScriptファイルを修正し、Gitを通じてデプロイします。
